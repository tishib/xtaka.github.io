# 天気 / Weather App

日本の天気予報をターミナル風UIで表示するシングルページアプリです。

## 機能

- 現在地を自動検出して都道府県を選択
- 今日の天気・気温（最低/最高）を表示
- 6時間ごとの降水確率をバーグラフで表示
- 週間予報（天気・気温・降水確率）
- 天気に応じたテーマカラーとアイコンアニメーション
- 47都道府県に対応

## 使い方

ローカルサーバーを起動してブラウザで開きます（`file://` では位置情報が動作しません）。

```bash
python3 -m http.server 8080
```

`http://localhost:8080` にアクセスしてください。

## データソース

| サービス | 用途 |
|----------|------|
| [気象庁 Forecast API](https://www.jma.go.jp/bosai/forecast/) | 天気予報データ |
| [Nominatim (OpenStreetMap)](https://nominatim.openstreetmap.org/) | 座標 → 都道府県の変換 |

## 技術仕様

- 単一の `index.html` のみ（依存ライブラリなし）
- ビルド不要
